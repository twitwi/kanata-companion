# kanata-companion

This template should help get you started developing with Vue 3 in Vite.

## Recommended IDE Setup

[VS Code](https://code.visualstudio.com/) + [Vue (Official)](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur).

## Install for build

You need:
- pnpm (or just npm)
- tauri.

For npm/pnpm, the best way depends on your setup, https://pnpm.io/installation .

For tauri, see https://tauri.app/start/prerequisites/ .

Run:

~~~
pnpm install
~~~


## Build

Run:

~~~
pnpm tauri dev
~~~

Build app and packages

~~~
pnpm tauri build
#pnpm tauri bundle
~~~
